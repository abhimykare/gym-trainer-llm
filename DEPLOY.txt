STEP 4: Add Persistent Disk
----------------------------
1. Scroll to "Disk" section
2. Click "Add Disk"
3. Name: whatsapp-session
4. Mount Path: /app/whatsapp-session
5. Size: 1 GB

STEP 5: Deploy
--------------
1. Click "Create Web Service"
2. Wait 15-20 minutes (Docker build takes time)
3. Watch logs - you'll see Docker building Chrome
4. Look for QR code in logs

-------------------------
1. View logs in Render dashboard
2. Find QR code (ASCII art)
3. Open WhatsApp on phone (8848091098)
4. Settings → Linked Devices → Link a Device
5. Scan QR code
6. Session saves to disk!

TROUBLESHOOTING:
----------------
If you see "Running 'node src/index.js'" in logs:
→ You selected Node environment instead of Docker!
→ Delete service and recreate with Docker environment

If you see Docker build logs:
→ ✅ Correct! Wait for build to complete
